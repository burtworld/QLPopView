## Troduce
	这是看了别人写的代码，然后自己拼合起来的
    iOS二维码工具包，包含功能
    1.	扫描识别二维码
    2. 生成二维码图片（添加圆角，图片）
    3. 识别图片中的二维码

## Installation

QLQRCodeUtils is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'QLQRCodeUtils'
```
